Proyecto ejemplo: localStorage

Este proyecto compara una variable normal de JavaScript, sessionStorage y localStorage.

1. Variable JS

Esta variable vive solo mientras la página está cargada. Si recargas, vuelve a cero.

Contador temporal: 0

2. sessionStorage

Este dato permanece al recargar, pero desaparece al cerrar la pestaña.

Nota guardada: Nada todavía

3. localStorage: lista de tareas persistente

Estas tareas permanecen aunque cierres y vuelvas a abrir el navegador.

    4. fetch + localStorage

    Este botón pide un usuario a una API de prueba y lo guarda localmente. Luego puedes reutilizarlo sin hacer otra petición. 

    No hay usuario cargado.

    Recordatorio de seguridad

    No guardes contraseñas, tokens sensibles ni datos personales delicados en localStorage, porque cualquier JavaScript de la página puede leerlos.
